Technologies Used

  • Angular: Front-end framework used for building the user interface with a focus on responsive and dynamic user experiences.
  • Nest.js: Back-end framework that provides a robust and scalable architecture for handling the application's business logic and API endpoints.
  • PostgreSQL: Relational database management system used for storing and querying user data securely and efficiently.
  • Flask: Lightweight Python web framework used for implementing machine learning models and serving AI functionalities.
  • AWS: Amazon Web Services for hosting, storage, and various cloud-based services ensuring scalability and reliability of the application.
  • Redis: In-memory data structure store used for caching and ensuring fast data access and efficient inter-service communication.
  • TypeORM: ORM framework for TypeScript and JavaScript, used with Nest.js to interact with the PostgreSQL database.
  • TensorFlow & PyTorch: Machine learning libraries used for developing and deploying AI models.
  • Docker: Containerization platform used for deploying and managing applications in isolated environments.
  • Git: Version control system used for tracking changes in the codebase and facilitating collaboration among team members.

These technologies were chosen to ensure a robust, scalable, and efficient development and deployment process, leveraging modern frameworks and cloud services.
