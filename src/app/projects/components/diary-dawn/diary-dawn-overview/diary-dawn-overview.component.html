Diary Dawn Overview

Diary Dawn is a cutting-edge web application designed to help users gain insights into their personal growth and development through the power of machine learning and artificial intelligence. Built with a robust tech stack, including Angular for the front end, Nest.js for handling general application logic, Flask for machine learning tasks, PostgreSQL for the database, and AWS for hosting, Diary Dawn exemplifies modern web development practices.

Key Features:

  • Intelligent Diary Entries: Users can create diary entries that are analyzed by advanced AI algorithms to provide personalized insights.
  • User-Friendly Interface: A sleek and intuitive design ensures a seamless user experience, making it easy for anyone to navigate and utilize the app's features.
  • Privacy and Security: With a strong emphasis on user privacy, all data is securely stored and processed, ensuring that personal information remains confidential.
  • Machine Learning Classifiers: The app employs sophisticated ML classifiers through a Flask backend to analyze text entries, offering users valuable perspectives on their writing.
  • Integrated ChatGPT: Diary Dawn integrates ChatGPT in two ways: a publicly available version for anyone to try, and an enhanced version for authorized users that helps explain results and provides additional support.

Diary Dawn is more than just a project; it is a testament to the power of modern technology in enhancing personal development. The journey of creating this application involved countless hours of coding, testing, and refining, supported by a dedicated setup of multiple computers working in harmony.
